## 2. 画面仕様

---

### SCR-01：ホーム画面

**表示要素**
- アプリタイトル・キャッチコピー
- 本日の進捗サマリー（学習済み枚数 / 目標枚数）
- 単語ステータスの内訳（new / learning / mastered）
- 「学習スタート」ボタン → SCR-02 へ遷移
- 「単語を管理」ボタン → SCR-04 へ遷移

**ロジック**
- `toeic_progress.date` が本日と異なる場合、`studied` と `correct` を 0 にリセット
- 登録単語が 0 件の場合、「単語を追加してください」のガイドを表示

---

### SCR-02：学習画面（カード）

**表示要素**
- 進捗バー（現在枚数 / セッション全枚数）
- フリップカード
  - 表面：英単語、品詞
  - 裏面：日本語の意味、英語例文、日本語例文訳
- 「もう一度」ボタン（赤系）
- 「覚えた！」ボタン（緑系）
- 「管理画面へ」リンク

**操作仕様**
| 操作 | 動作 |
|------|------|
| カードをタップ/クリック | 表裏フリップ（CSSアニメーション） |
| 「覚えた！」ボタン | `correct_count +1`、次のカードへ |
| 「もう一度」ボタン | `miss_count +1`、セッション末尾に再挿入 |
| キーボード `→` | 「覚えた！」と同じ |
| キーボード `←` | 「もう一度」と同じ |
| キーボード `Space` | カードフリップ |

**カード出題順**
- 設定が `"priority"` の場合：優先度スコア昇順（データ設計書参照）
- 設定が `"random"` の場合：シャッフル
- `status: "mastered"` の単語はデフォルトで除外

**ロジック**
- カードが全て完了 → SCR-03 へ自動遷移
- `localStorage` へのデータ保存はボタン押下のたびにリアルタイム実行

---

### SCR-03：セッション完了画面

**表示要素**
- 完了メッセージ（例：「お疲れさまでした！」）
- セッション結果
  - 学習枚数
  - 正解率（覚えた ÷ 全出題）
  - 苦手認定された単語数（miss_count が増加した単語）
- アクションボタン
  - 「もう一度（苦手のみ）」→ miss が1以上の単語のみで再セッション
  - 「ホームへ戻る」→ SCR-01

---

### SCR-04：単語管理画面

**表示要素**
- 単語一覧テーブル（単語 / 品詞 / 意味 / カテゴリ / ステータス / 操作）
- 検索フィールド（単語名・意味でインクリメンタルサーチ）
- カテゴリフィルター（ドロップダウン）
- 「＋ 新しい単語を追加」ボタン → 追加フォームを展開

**追加・編集フォーム項目**
| フィールド | 型 | 必須 | 備考 |
|-----------|-----|------|------|
| 英単語・熟語 | text | ✓ | 重複チェックあり |
| 品詞 | select | ✓ | v. / n. / adj. / adv. / phrase |
| 日本語の意味 | text | ✓ | |
| 英語例文 | textarea | | |
| 例文の日本語訳 | textarea | | |
| カテゴリ | text | | 例: Part7, 金の読解 |

**削除仕様**
- 削除ボタン押下で確認ダイアログを表示
- 確認後に `toeic_words` から該当 ID を削除

## 4. エラー処理

| ケース | 対応 |
|--------|------|
| localStorage が使用不可 | アラートで通知し、セッション中のみデータを保持 |
| 登録単語が 0 件でスタートボタン押下 | 「単語が登録されていません」メッセージを表示 |
| フォームの必須項目が未入力 | 該当フィールドに赤いボーダーとエラーメッセージを表示 |
| 重複単語の登録 | 「すでに登録されています」と警告、登録は行わない |
